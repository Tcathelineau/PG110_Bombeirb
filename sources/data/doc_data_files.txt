** Documentation sur le format des fichiers "data" :

* Games default
Structure : "game_default_<mode>.txt" pour le fichier par défaut
			"game_saved.txt" pour le fichier de la partir en cours

Contenu : 
		1ère ligne : nombre total de niveaux
		2ème ligne : préfixe des cartes

* Maps
Structure : "map_<prefix>_<level>.txt
Contenu : 
		1ère ligne : taille de la carte
		2ème ligne : coordonnées d'apparition sur la carte
		ensuite : grille des cases avec gestion des surplus d'espaces, bien laisser un espace après chaque type
